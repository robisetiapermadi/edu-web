Tutorial Membuat Tabel pada HTML

August 4, 2024

#html

Hallo kali ini saya menulis tutorial membuat tabel pada HTML. Sebelum ke tutorial, saya akan menjelaskan dulu apa itu tabel.

Pengertian

Membuat tabel pada HTML adalah salah satu dasar yang penting dalam pembuatan halaman web. Tabel digunakan untuk menyajikan data dalam format yang terstruktur dan mudah dibaca. Berikut adalah tutorial lengkap tentang cara membuat tabel pada HTML.

Struktur Dasar Tabel HTML

Tabel HTML dibuat menggunakan tag <table> dan memiliki beberapa elemen penting:

  • <table>: Tag utama untuk membuat tabel.
  • <tr>: Tag untuk membuat baris dalam tabel.
  • <th>: Tag untuk membuat sel header dalam tabel, biasanya digunakan untuk judul kolom.
  • <td>: Tag untuk membuat sel data dalam tabel.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Contoh Tabel HTML</title>
</head>
<body>

<table border="1">
    <tr>
        <th>Nama</th>
        <th>Umur</th>
        <th>Kota</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>25</td>
        <td>Jakarta</td>
    </tr>
    <tr>
        <td>Bob</td>
        <td>30</td>
        <td>Bandung</td>
    </tr>
</table>

</body>
</html>
